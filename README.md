# Cypress Law Group – WordPress Demo

This repository contains the core WordPress theme and custom code used to build the **Cypress Law Group** website demo. It includes the full `wp-content` directory, which holds all design assets, templates, plugins, and custom functionality for the site.

##  Project Overview
Cypress Law Group is a clean, modern legal website concept built in WordPress.  
It features:

- Appointment scheduling features  
- Custom styling and page templates  
- Simple, easy-to-navigate structure  
- ‘About’, ‘Contact’ and ‘Dashboard’ pages  

##  What’s Included
- `wp-content/themes/` – Theme files used for layout, design, and structure  
- `wp-content/plugins/` – Any active plugins used for functionality  
- Custom CSS, JS, and PHP template files  
- Other WordPress-related assets used in the demo  

*(The full WordPress installation is intentionally excluded for size and security reasons.)*
